# 🔐 LinkShield – URL Threat Detection System

## Overview
LinkShield is a rule-based URL safety checker that detects suspicious URLs using weighted scoring and structural analysis.

## Features
- HTTPS validation
- Suspicious keyword detection
- Special character detection
- Risk scoring (0–100%)
- Safe / Suspicious / High Risk classification
- Animated cyber dashboard UI

## Technologies Used
- Python (Flask)
- HTML
- CSS
- JavaScript
- Gunicorn
